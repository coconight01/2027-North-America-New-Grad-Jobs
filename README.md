## Data

- `config/sources.yml`: ATS boards and GitHub discovery feeds.

## Automatic updates

The workflow runs every six hours and:

1. Fetches public Greenhouse, Lever, Ashby, SmartRecruiters, and configured GitHub job-list feeds.
2. Filters for North American full-time 2027 new-grad roles.
3. Normalizes categories and locations.
4. Deduplicates by canonical application URL and normalized role identity.
5. Regenerates CSV, JSON, and this README.
6. Commits only when the dataset changes.

Run locally:

```bash
python -m pip install -e .
python scripts/update_jobs.py
```

Include general new-grad jobs that do not explicitly mention 2027:

```bash
python scripts/update_jobs.py --include-general
```
